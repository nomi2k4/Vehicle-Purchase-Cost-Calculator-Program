Vehicle Purchase Cost Calculator Program:

This C++ program was written to calculate the total cost of purchasing a vehicle, including various taxes.

Program Overview:

- Prompts the user to input the price of the vehicle
- Calculates the following taxes:
    - Sales tax (6.5% of the vehicle price)
    - Delivery tax (0.5% of the vehicle price)
    - Property tax (1% of the vehicle price)
- Calculates the total amount including all taxes
- Displays the result

Features:

- User input for vehicle price
- Use of multiplication and division operators for tax calculations
- Clear and informative output

Assumptions:

- Sales tax rate: 6.5%
- Delivery tax rate: 0.5%
- Property tax rate: 1%

How to Run the Program:

1. Copy the provided C++ code into a .cpp file
2. Compile the program using a C++ compiler
3. Run the compiled program:
    - On Linux/macOS: ./vehicle_purchase_cost_calculator
    - On Windows: vehicle_purchase_cost_calculator.exe
4. Enter the vehicle price when prompted

Learning Objectives:

- Understanding basic user input and output
- Using floating-point variables and arithmetic operators
- Working with basic calculations and tax rates
- Basic program structure and syntax
